3️⃣  CRÉER UN TOKEN GITHUB (5 minutes)

a) Ouvrir dans votre navigateur :
   https://github.com/settings/tokens

b) Cliquer sur "Generate new token" → "Generate new token (classic)"

c) Remplir :
   Note : Ghost Remix Pack Deploy
   Expiration : 90 days
   Cochez : ☑ repo (toute la section)

d) Cliquer sur "Generate token" (en bas)

e) COPIEZ LE TOKEN (il commence par ghp_xxxxx)
   ⚠️ Il ne sera plus visible après !

f) Retournez dans votre terminal

g) Quand Git demande "Password", COLLEZ LE TOKEN
